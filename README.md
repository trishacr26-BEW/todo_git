CLI To-Do Task Manager 📝

A simple Command Line To-Do Task Manager built using Python.
This project allows users to manage their daily tasks directly from the terminal.

🚀 Features

- Add a new task
- View all tasks
- Delete a task
- Simple menu-based interface

📂 Project Structure

todo-manager
│
├── tasks.py        # Stores the task list
├── add_task.py     # Function to add tasks
├── view_tasks.py   # Function to display tasks
├── delete_task.py  # Function to delete tasks
├── menu.py         # Displays menu options
└── main.py         # Runs the main program

⚙️ Requirements

- Python 3.x
- VS Code or any Python IDE

▶️ How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open the terminal in the project directory.
4. Run the program:

python main.py
